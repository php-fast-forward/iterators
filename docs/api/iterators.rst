Iterators
=========

.. list-table:: Core Iterators
   :header-rows: 1

   * - Class
     - Description
   * - ``ChainIterableIterator``
     - Chains multiple iterables into a single iterator.
   * - ``ChunkedIteratorAggregate``
     - Splits data into fixed-size chunks.
   * - ``ClosureIteratorIterator``
     - Wraps an iterator with a closure for custom logic.
   * - ``ConsecutiveGroupIterator``
     - Groups consecutive elements by a callback.
   * - ``FileExtensionFilterIterator``
     - Filters files by extension.
   * - ``GeneratorCachingIteratorAggregate``
     - Caches generator results for repeatable iteration.
   * - ``GeneratorRewindableIterator``
     - Allows rewinding generators.
   * - ``GroupByIteratorIterator``
     - Groups elements by a key or callback.
   * - ``InterleaveIteratorIterator``
     - Interleaves multiple iterators.
   * - ``IterableIterator``
     - Adapts any iterable to an iterator.
   * - ``LookaheadIterator``
     - Supports lookahead/lookbehind while iterating.
   * - ``RangeIterator``
     - Iterates over a numeric range.
   * - ``RepeatableIteratorIterator``
     - Makes any iterator repeatable.
   * - ``SlidingWindowIteratorIterator``
     - Iterates with a sliding window over data.
   * - ``TrimIteratorIterator``
     - Trims elements from the start/end of an iterator.
   * - ``UniqueIteratorIterator``
     - Filters out duplicate values.
   * - ``ZipIteratorIterator``
     - Zips multiple iterators together.

See each class for constructor details and usage examples.
