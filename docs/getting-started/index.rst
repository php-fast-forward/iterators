Getting Started
===============

This section helps you go from installation to your first practical iterator
pipeline with as little context-switching as possible.

You will learn:

- how to install the correct Composer package;
- what kind of values the library accepts as input;
- what a minimal chunking workflow looks like;
- where to go next once you are comfortable with the basics.
